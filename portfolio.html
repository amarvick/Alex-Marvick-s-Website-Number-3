BANKING GAME

In this game, two different classes are used; one being the program itself that imports the other class, where all of the functions are represented. The computer prompts you that you have $10000, and it asks how much of it you'd like to store in your checking and savings account with the remainder in your pockets. The computer will continually give you a menu asking what you'd like to do with your money; either make a deposit, withdrawal, transfer between accounts, check balances, or quit. There are some other options as well included that haven't been edited yet (you'll notice a luck option and an option to check your items), and I'm not sure how I'm going to make those parts but it's there for making the game more interesting. It's great extra practice, too.

Check it out on Github!

This was a fun project I worked on with my brother on how to use the Reddit API via JavaScript. For example, if you type 'aww' in the search box, JavaScript will find the first four most recent 'aww' articles and post their names, and can do this for any of the other headings searched.

I think this will be the most appealing for any user who is just looking to play a fun game. This game has many different random words that could possibly be pulled out, and you can guess incorrectly up to 8 times before you become hung. This was done by using the basic concepts of vectors, while loops and if/else statements.

Check it out on Github!

The MasterQuest

Perhaps one of the more fun projects I've worked on, this game requires the user to log in to the server to begin. There are only a few accounts that I've made available for people; therefore, you will have to sign in as a 'guest'. This won't impact the journey you'll be taking, however. All you have to do is input 'guest' for either the username or the password. This portion of the program uses the fundamentals of logical operators (&&, || and !=), while the actual journey that you'll be taking uses a combination of do loops, if/then statements, switch/case options as well as break/continue statements. There is a maximum possible score in this game -- if you play this and let me know what that maximum score is, that's cool!

Check it out on Github!

CYLINDRICAL & SPHERICAL COORDINATES CONVERSION CALCULATOR

In Calculus, you have the ability to take regular cartesian coordinates (x, y and z on a 3-dimensional plane) and convert them in to Cylindrical and Spherical Coordinates. In both of these coordinate systems, you can think of creating a coordinate point on the edge of a Cylinder or a Sphere. To see how the calculations actually work, you'd have to read the code -- but if you're doing this unit in Calculus III and need to check your answers, feel free to do so with this program. Created: January 2015 (My first simple formal program made)

Check it out on Github!
